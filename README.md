# 🛍️ Product Explorer

A modern and responsive Product Explorer web application built with **React**, **Vite**, and **Tailwind CSS**. This application fetches product data from the FakeStore API and provides a clean, user-friendly shopping experience.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/product-explorer.git
```

### Navigate to the project

```bash
cd product-explorer
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

---

## 🔧 Environment Variables

Create a `.env` file in the project root.

```env
VITE_API_URL=http://localhost:5000
```
